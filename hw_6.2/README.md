# Задание 6.1 - Prop descriptor

Реализовать дескриптор prop aka `property` в виде декоратора. ​

Пример использования:​

```python
class Something:​
    def __init__(self, x):​
        self.x = x​

    @prop​
    def attr(self):​
        return self.x ** 2​

    @attr.setter
    def attr_setter(self, update):
        return self.x = update
​
>>> s = Something(10)​
>>> s.attr​
100​
>>> s.attr​ = 3
>>> s.attr​
9
```
